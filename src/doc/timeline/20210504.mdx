# 두번째 Mindustry JAVA 모드, Informatis 개발 시작

---

Informatis는 콘텐츠 모드였던 Sharustry와 달리 유틸리티 모드로써 버전 호환성을 가진 모드입니다.  
기본 바닐라 기능으론 절대 알 수 없는 정보부터, 압도적 편의성 및 유틸리티 기능 제공을 통해 UX 향상을 목적으로 뒀습니다.  
다른 모드들도 그렇지만 확고한 기획, 의지, 인기로 동기와 구체화가 모두 이뤄져 제 개발사 초기에 부스트를 달아준 모드였습니다. 커뮤니티 덕이 있겠지만 아래 레포지토리의 별 수는 제 동기가 되기에 충분했으리라 믿어 의심치 않습니다.  
libGDX의 UI 구조는 Unity와 달리 런타임에 UI를 생성하고 삭제하며 특이하게도 레이아웃 구조가 박스 모델입니다. 이러한 박스 모델의 align, padding, margin 구성요소들을 익힘으로써 미래에 시작한 웹개발에 아주 큰 도움이 되었습니다.
