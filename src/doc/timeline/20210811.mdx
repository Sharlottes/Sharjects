# 첫 안드로이드 앱, Timer 개발 시작

---

Timer는 간단한 알람 & 스톱워치 안드로이드 네이티브 어플리케이션입니다.  
Informatis로 박스 모델링을 경험해본 저는 자신있게 안드로이드 xml 레이아웃에 도전했습니다.
하지만 메이저 분야는 어느 마이너하고 좁은 분야들과 달리 너무 방대하고 복잡했습니다. 헤딩으로 일관하기엔 힘든 부분들이 많았고, 그걸 간과한 전 얼마 지나지 않아 지쳐 떨어져나갔습니다.
결과적으로 저는 안드로이드 개발을 하면서 머터리얼 디자인과 GUI 레이아웃 편집을 습득하긴 했습니다.
