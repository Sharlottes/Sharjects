# JudgeKit 개발

---

judgekit는 제 첫번째 NPM CLI 모듈 프로젝트입니다.
백준 문제를 풀어나가며 반복적인 테스트와 코드 생성에 지친 저는 이 모든걸 자동화할 Node.js 스크립트를 만들었고, 이를 전역적으로 실행하고 싶어 NPM CLI로 확장하였습니다.

<RepoCard username="sharlottes" repository="judgekit" />

CLI는 분명 새로운 분야고 미지의 세계였습니다. 하지만 전 아랑곳하지 않고 일주일만에 100커밋과 8스타, 약 600다운로드 등 단편적으로 놀라울만한 결과를 내놓아 저 자신을 놀라게 했습니다.
그러나 일주일이 지나고 나니 금세 다시 지쳐버렸습니다. 전 기획이 있던 일주일과 기획이 없던 일주일을 연속하여 경험한 것입니다.

전 Timer로 존재하지 않는 기획을 경험했고, 여러 프로젝트들로 만들면서 생겨나는 기획을 경험했으며, Dino로 절반짜리 기획을 경험했고, JudgeKit로 있다가 사라진 기획을 경험했으며 PrincessSavior로 완벽한 기획을 목격했습니다.
이렇게나 다양한 경험들은 한가지 명백한 명제를 가리키고 있었습니다, **기획은 프로젝트의 시작이자 반**이라는 것을.
