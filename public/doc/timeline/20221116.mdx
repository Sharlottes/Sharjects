# TimerWeb 개발

---

timerWeb는 [수능 디데이 타이머](https://jinh.kr/SAT_kr/)에서 영감을 받아 만든 유틸리티 타이머 임베드 생성기이며 경량화를 목적으로 MUI와 여타 모듈을 모두 배제하고 css, react, next로만 구성된 가벼운 웹사이트입니다.  
kakaoNacksee에 이은 두번째 미니 웹 프로젝트인데, 이번 개발에선 바닐라 css를 경험하며 여러 스타일 라이브러리의 필요성을 깨닫게 되었습니다.  
next를 선택한 이유는 vercel이 next만을 받아들인다고 생각했기 때문인데 vite도 가능하단 사실을 알아서 곧 바꿔야 할 것 같습니다.
